# WideoClient
##这是什么
* 这是个练习作品。
* 一个视频分享软件，用户可以上传视频，观看他人上传的视频。
* UI基于[MDUI](http://mdui.org/)
* 功能实现来自[MUI](http://www.dcloud.io/mui.html)。
* API接口加密使用[CyptoJS](https://github.com/brix/crypto-js)
